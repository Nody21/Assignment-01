No. Tag Name Description
01 !DOCTYPE Defines the document type
02 acronym Defines an acronym
03 head Defines the document type
04 body Defines the document's body
05 div Defines a section in a document
06 article The
element identifies a self-contained piece of content which could theoretically be distributed to other websites and platforms as a stand-alone unit. The
element is a good choice to contain entire blog posts, news articles, and similar content.
07 title The element is a required HTML element used to assign a title to an HTML document. Page titles are not displayed in the browser window, but they are used as the page name by search engines and displayed by browsers in the title bar, on the page tab, and as the page name of bookmarked webpages.
08 img Defines an image
09 link The element is used to define a relationship between an HTML document and an external resource. This element is most commonly used to define the relationship between a document and one or more external CSS stylesheets.
10 li The element defines a list item that is part of an ordered and unordered list of items.
11 menu The element defines an instance of a menu. This experimental HTML feature has very limited browser support, but may soon be an effective way to add menu items to context menus and to create interactive web application menus.
12 ol The element is used to create an ordered list. An ordered list is created by nesting one or more elements between the opening and closing
    tags.
13 style Defines style information for a document
14 p The element is used to identify blocks of paragraph text. The closing tag is optional and is implied by the opening tag of the next HTML element encountered in an HTML document after an opening.
15 table The element is used in conjunction with child elements and others to add tabular data to an HTML document.
16 i The element is used to differentiate words from the surrounding text by styling the marked text in italics without implying any added emphasis to the italicized words.
17 br The element is used to insert a line break or carriage-return within a parent element such as a paragraph without breaking out of the parent container.
18 big The element was used to cause the selected text to appear one size larger than the surrounding text. This purely presentational tag was removed from HTML5 and should not be used. Instead, use CSS to control font size.
18 data Links the given content with a machine-readable translation
19 col Specifies column properties for each column within a element
20 audio Defines sound content
21 type The element was used to create a browser-specific "Easter Egg" effect. It was never added to the official HTML specification and is no longer supported by any browser. Do not use it.
22 meta Defines metadata about an HTML document
23 track The element is used as a child of an element and adds a time-based data source to the parent media element. For example, the element can be used to add timed subtitles to a video and closed captions to audio.
24 abbr Defines an abbreviation or an acronym
25 b Defines bold text
26 del Defines text that has been deleted from a document
27 pre Defines preformatted text
28 rp Defines what to show in browsers that do not support ruby annotations
29 rt Defines explanations and pronunciations of characters in ruby language annotations (a system for displaying certain characters in several Asian languages).
30 q The element is used to identify and inline quote that does not require paragraph breaks. Longer quotations that do require paragraph breaks should use the
element.

element is used to identify content that is related to the primary content of the webpage, but does not constitute the primary content of the page. Author information, related links, related content, and advertisements are exampes of content that may be found in an aside element.

33 small Defines smaller text
34 h1 to h6 Defines HTML headings
35 code Defines a piece of computer code
36 meter Defines a scalar measurement within a known range (a gauge)

38 var The element is used to identify a variable in a mathematical equation or computer program. Text marked with tags is displayed in an italics font style by most browsers.

49 lebel Defines a label for an element
50 option Defines an option in a drop-down list
51 dialog The element is used to create a dialog box such as a popup or modal window that is rendered within the active browser window. Defining the behavior of a element typically requires the use of JavaScript.
52 main The element is used to denote the content of a webpage that relates to the central topic of that page or application. It should include content that is unique to that page and should not include content that is duplicated across multiple webpages, such as headers, footers, and primary navigation elements.
53 time Defines a datetime. The content of a time element is typically a human-readable date and time, and a machine-readable version of the same time is placed in the datetime attribute.
